Installation et Configuration de Git

1. Installation de Git

Sur Windows

  1. Télécharger Git

    2. Rendez-vous sur git-scm.com et téléchargez l’installateur pour Windows.

  2. Lancer l'installation

    3. Exécutez le fichier .exe téléchargé.

    Acceptez les termes et conditions.

    Options importantes à choisir pendant l’installation :

    • Editor par défaut : Choisissez VS Code ou un autre éditeur si nécessaire.
    • PATH Environment : Sélectionnez Git from the command line and also from 3rd-party software.
    • Line Ending Conversion : Sélectionnez Checkout Windows-style, commit Unix-style line endings.

  3. Vérifier l’installation

    4. Ouvrez Git Bash et exécutez : 

    									
										git --version

Sur macOS

  1. Vérifier si Git est déjà installé

    2. Ouvrez un terminal et exécutez : git --version

    Si Git est installé, vous verrez une version affichée.

  2. Installer Git via Homebrew (recommandé)

    3. Installez Homebrew si ce n'est pas encore fait : /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

    Installez Git : brew install git

    Vérifiez l’installation avec : 

    										
											git --version

  3. Installer Git manuellement (alternative)

    4. Téléchargez le package depuis git-scm.com et suivez l'installation.

Sur Linux (Debian/Ubuntu, Fedora, Arch)

Sur Debian/Ubuntu :

									
										sudo apt update
										sudo apt install git

Sur Fedora :

									
										sudo dnf install git

Sur Arch Linux :

									
										sudo pacman -S git

Vérifier l’installation :

									
										git --version

2. Configuration initiale de Git

Configurer le nom et l'email

							
								git config --global user.name "Votre Nom"
								git config --global user.email "votre.email@example.com"

Vérifier la configuration actuelle

							
								git config --list

Configurer l'éditeur par défaut

Si vous voulez utiliser VS Code comme éditeur :

							
								git config --global core.editor "code --wait"

3. Génération et ajout d’une clé SSH (optionnel mais recommandé)

Générer une clé SSH

							
								ssh-keygen -t rsa -b 4096 -C "votre.email@example.com"

  • Appuyez sur Entrée pour enregistrer dans le dossier par défaut (~/.ssh/id_rsa).
  • Entrez (ou non) une phrase secrète.

Ajouter la clé SSH à GitHub

  1. Afficher la clé SSH générée :
    2. 							
								cat ~/.ssh/id_rsa.pub
  2. Copiez la clé et ajoutez-la sur GitHub :
    • Allez sur GitHub > Settings > SSH and GPG keys.
    • Cliquez sur New SSH Key et collez la clé.
  3. Tester la connexion :
    4. 								
									ssh -T git@github.com
    • Si tout fonctionne, vous verrez :"Hi votre_nom! You've successfully authenticated."

4. Premiers pas avec Git

Initialiser un dépôt Git

						
							mkdir mon_projet
							cd mon_projet
							git init

Ajouter et valider des fichiers

echo "Hello Git!" > readme.txt ajout readme.txt commit "Premier commit"

Lier un dépôt GitHub

						
							git remote add origin git@github.com:votre_user/mon_projet.git
							git push -u origin main
